# 🎓 API DOCUMENTATION

## 📋 Overview
API untuk mengelola data Program Khusus sekolah yang mencakup program ICT dan Tahfidz dengan informasi lengkap tentang fitur, benefit, target audience, dan pengaturan tipe program.

## 🔗 Base URL
```
http://api.raphnesia.my.id/api/v1/program-khusus
```

---

## 🔧 Error Responses

### **400 Bad Request**
```json
{
  "success": false,
  "message": "Tipe program tidak valid"
}
```

### **404 Not Found**
```json
{
  "success": false,
  "message": "Program tidak ditemukan",
  "error": "No query results for model [App\\Models\\SpecialProgram] 999"
}
```

### **500 Internal Server Error**
```json
{
  "success": false,
  "message": "Gagal mengambil data program khusus",
  "error": "Database connection failed"
}
```

## 🔒 Authentication

Saat ini API bersifat **public** dan tidak memerlukan autentikasi. Semua endpoint dapat diakses langsung tanpa token atau kredensial.

---

## 📝 Notes

1. **Image URLs**: Semua URL gambar menggunakan path relatif. Untuk akses penuh, tambahkan base URL: `http://localhost:8000/storage/`

2. **Formatted Fields**: Model menyediakan accessor untuk format yang user-friendly:
   - `formatted_price`: Format harga dalam Rupiah
   - `formatted_duration`: Format durasi dalam bulan

3. **Active Status**: Semua endpoint hanya mengembalikan data dengan status `is_active = true`

4. **Ordering**: Program diurutkan berdasarkan `sort_order` secara ascending

5. **Error Handling**: Semua endpoint menggunakan try-catch dan mengembalikan response JSON yang konsisten
